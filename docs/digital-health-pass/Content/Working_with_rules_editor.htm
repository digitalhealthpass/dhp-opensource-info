Working with the rules editor

Use the rules editor in the administration console to manage business rules in verifier configurations. Use configurations to reuse rules and customize their usage to meet your requirements. When you edit a configuration, you can modify rule values, and either enable or disable rules and specifications. After updating, you do not need to re-issue related verifier credentials.

In the rules editor, you work in this hierarchy:

Categories > Specifications > Rules

where

Categories are standard items, for example, Generic Credential and Temperature Scan.

Specifications are credential standards, for example, Digital Health Pass and EU Digital COVID Certificate.

Rules control how credentials are processed and assessed, for example, Check the credential is not expired and Check for negative test result.

Configurations and value sets

In the rules editor, when you select an item (a specification configuration) from the Config Values tab, you'll see configuration values and related information.
