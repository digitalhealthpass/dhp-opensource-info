Version 2 (March 2022)

Verification rules

In administration, System Administrators can customize verification rules on behalf of a customer.

  • For a given credential specification, administrators can enable or disable individual rules.
  • Administrators can enable or disable an entire specification.
  • Values used by specific rules cannot be modified in a given configuration.

Note that only System Administrators can customize verification rules in Version 2. For more information, see the Administrator's Guide.

Verifier metrics

System Administrators and Customer Administrators now have the ability to export metering reports. Reports can be exported in comma-separated value (CSV), PNG, or JPG format. Administrators can export either the entire report (that is, all charts within a given report), or a single chart from a report.

Improved charts

To improve readability, uses bar chart as the standard format in administration.

Accessibility improvements

This administration release includes support for assistive (Section 508) technologies. The administrative user interface now supports screen readers, magnifiers, and high contrast mode.
