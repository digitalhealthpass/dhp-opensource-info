Deleting a verifier configuration

You can delete a verifier configuration when, for example, it is no longer used. Deletion is permanent.

Important: After deleting a verification configuration, delete any verification credentials that are associated with it. If such a credential is not deleted, then it will be considered invalid.

To delete a verifier configuration:

  1. Log in to the administrative console.
  2. On the home page, click the Set up configurations tile.
  3. On the Configuration catalog page:
    1. Click the <Company> tab.
    2. In the row for the configuration that you want to delete, under Actions, click the trash icon.
