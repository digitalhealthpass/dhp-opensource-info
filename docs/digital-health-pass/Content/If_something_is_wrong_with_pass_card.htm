If something is wrong with a pass, card

If you use the Verify app to scan a health pass or card and there is a problem, share with the pass holder the reason that the pass is invalid. Follow any additional policies or guidance for your business.

General guidance on how to proceed with card, pass problems
Problem Solutions
1. Technical problems  
The Verify app does not detect the QR code.

Try scanning the QR code again.

The holder might have a QR code that's not intended for healthcare use.
The type of QR code is not supported.

supports selected types of healthcare QR codes. For information, see About supported healthcare QR codes.

2. Policy-related problems  
The card, pass is for the wrong type of test. Follow your organization's policies.
The card, pass is expired. See above.
