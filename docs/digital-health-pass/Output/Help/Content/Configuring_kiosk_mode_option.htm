Configuring kiosk mode option

Turn on and configure the optional Digital Health Pass Verify kiosk mode feature. Digital Health Pass Verify uses per-device settings for kiosk mode. The feature is separately enabled and configured from Single App Mode (iOS app lock) on Apple devices, lock task mode on Android devices, or mobile kiosk lockdown solutions from other vendors. For information about how kiosk mode works, see Self-service verification using kiosk mode.

Note: When Digital Health Pass Verify is installed on a tablet, kiosk mode is turned on by default.

To turn on kiosk mode:

  1. Make sure that the device is configured appropriately for your physical setting, for example, display brightness and orientation.
  2. Open the Verify app and open the app settings.
  3. Tap Kiosk.
  4. On the Kiosk page, enable Kiosk Mode.
  5. Configure kiosk mode settings as needed:
    1. Auto Dismiss Duration: Time period to display scan results before being automatically closed, in seconds.
    2. Always Auto Dismiss: Automatically close displayed scan results, whether the credential is verified or not verified.
    3. Default to Front Camera: Use the device's front camera for scanning. Disable this to use the rear camera instead.
  6. Test credential scanning to make sure that kiosk mode works as you've configured it.

To turn off kiosk mode:

  1. On Android devices, double tap toward the bottom of the screen.
  2. Open the Verify app and open the app settings.
  3. Tap Kiosk.
  4. On the Kiosk page, disable Kiosk Mode.
